# Personalized Workout & Diet Planner with AI

This project is an AI-based system that generates personalized workout routines and diet plans for students based on their individual needs.

Features:
- Customized workout plans
- Personalized diet suggestions
- Budget-based meal planning
- Student-friendly recommendations
- Web-based interactive interface

Technologies Used:
- Python
- Streamlit
- AI-based recommendation logic

How to Run:
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

Future Enhancements:
- Machine learning model integration
- Mobile app development
- Wearable device connectivity
